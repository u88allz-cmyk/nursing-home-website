# 상담 신청 관리 시스템 사용 가이드

## 📋 시스템 개요

해와달 요양원 웹사이트의 상담 신청 시스템은 다음과 같이 작동합니다:

### 고객이 상담 신청할 때:
1. **웹사이트 접속** → 상담문의 페이지로 이동
2. **온라인 상담 폼 작성**:
   - 이름 입력
   - 연락처 입력 (한국 전화번호 형식 자동 검증)
   - 문의 내용 작성
   - 개인정보 수집 동의 체크
3. **"상담 신청하기" 클릭**
4. **데이터베이스에 자동 저장** ✅
5. **고객에게 성공 메시지 표시**

### 요양원에서 상담 신청을 확인하는 방법:

## 🔐 관리자 페이지 접속

### 1단계: 관리자 페이지 접속
- **URL**: `https://your-domain.replit.app/admin`
- **로그인 비밀번호**: `haewadal2025`

### 2단계: 상담 신청 목록 확인
로그인 후 다음 정보들을 확인할 수 있습니다:

#### 📊 대시보드 통계
- **총 상담 신청 건수**
- **오늘 접수된 신청 건수**
- **이번 주 접수된 신청 건수**

#### 📝 상담 신청 상세 정보
각 상담 신청마다 다음 정보가 표시됩니다:
- **신청 번호** (자동 생성)
- **신청자 이름**
- **연락처** (클릭하면 바로 전화 연결)
- **문의 내용** (전체 텍스트)
- **신청 일시** (년-월-일 시:분:초)

### 3단계: 데이터 관리

#### 📥 CSV 다운로드
- **"CSV 다운로드"** 버튼 클릭
- **Excel에서 열어서 관리 가능**
- **파일명**: `상담신청목록_YYYYMMDD.csv`

#### 📞 고객 연락
- 연락처를 클릭하면 바로 전화 앱 실행
- 또는 수동으로 연락처 복사하여 사용

## 💡 활용 방법

### 일일 업무 루틴
1. **매일 오전**: 관리자 페이지 접속하여 새 상담 신청 확인
2. **신청자에게 전화**: 연락처 클릭하여 바로 통화
3. **상담 진행**: 요양원 소개, 입소 절차 안내
4. **기록 관리**: 필요시 CSV 다운로드하여 별도 관리

### 주간/월간 분석
- **통계 데이터 활용**: 상담 신청 추세 파악
- **마케팅 효과 측정**: 특정 기간 신청 증가 분석
- **고객 관리**: CSV 데이터로 고객 DB 구축

## 🔒 보안 관리

### 비밀번호 변경 방법
현재 비밀번호를 변경하려면 개발자에게 문의하세요.

### 접근 권한
- 관리자 페이지는 비밀번호로 보호됩니다
- 고객 개인정보가 포함되어 있으므로 외부 유출에 주의하세요

## 📞 기술 지원

### 문제 발생 시
1. **페이지가 로드되지 않는 경우**: 인터넷 연결 확인
2. **로그인이 안 되는 경우**: 비밀번호 `haewadal2025` 정확히 입력
3. **데이터가 보이지 않는 경우**: 페이지 새로고침
4. **기타 기술적 문제**: 개발자 문의

### 백업 및 복구
- 모든 상담 신청 데이터는 자동으로 데이터베이스에 백업됩니다
- 실수로 데이터를 삭제할 위험이 없도록 설계되었습니다

## 📈 시스템 특징

### ✅ 장점
- **실시간 접수**: 고객이 신청하면 즉시 확인 가능
- **모바일 최적화**: 스마트폰에서도 편리하게 관리
- **데이터 보존**: 모든 신청 데이터 영구 보관
- **Excel 호환**: CSV 다운로드로 기존 업무와 연계

### 🚀 향후 개선 가능 기능
- 이메일 자동 알림 (새 상담 신청 시)
- SMS 자동 발송 (상담 완료 후 감사 메시지)
- 상담 상태 관리 (접수/진행중/완료)
- 고객 재방문 추적
