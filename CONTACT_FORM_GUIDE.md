# 상담 문의 폼 운용 가이드

## 현재 시스템
- **네틀리 폼스(Netlify Forms)** 사용
- 데이터베이스 없이 네틀리에서 직접 관리
- 실시간 이메일 알림 가능

## 상담 문의 확인 방법

### 1. 네틀리 대시보드 접속
1. https://app.netlify.com 로그인
2. "해와달 요양원" 사이트 선택
3. 좌측 메뉴에서 **"Forms"** 클릭

### 2. 문의 내용 확인
- 제출된 모든 상담 문의를 시간순으로 확인 가능
- 다음 정보가 수집됩니다:
  - **이름**: 상담 신청자 성명
  - **전화번호**: 연락처 (클릭 시 바로 전화 가능)
  - **상담 내용**: 구체적인 문의사항

### 3. 데이터 내보내기
- **CSV 파일 다운로드** 가능
- 엑셀에서 열어서 관리 가능
- 정기적으로 백업 권장

## 이메일 알림 설정 (권장)

### 네틀리에서 알림 설정
1. Forms 페이지에서 **"Settings"** 클릭
2. **"Form notifications"** 설정
3. 이메일 주소 입력 (예: haewadal@example.com)
4. 새 문의가 접수될 때마다 즉시 이메일 알림

## 문의 응답 방법

### 전화 응답
- 네틀리 대시보드에서 전화번호 클릭
- 모바일에서 바로 전화 연결
- 상담 내용 미리 확인 후 통화

### 문의 관리
- 처리 완료된 문의는 따로 표시 가능
- 중요한 문의는 별도 메모 추가
- 월별/주별 통계 확인 가능

## 장점
✅ **무료**: 월 100건까지 무료  
✅ **간편함**: 별도 데이터베이스 관리 불필요  
✅ **안전함**: 네틀리에서 보안 관리  
✅ **실시간**: 즉시 알림 가능  
✅ **접근성**: 어디서든 웹브라우저로 확인

## 문의 급증 시 대응
- 월 100건 초과 시 네틀리 유료 플랜 고려
- 또는 구글 폼스로 대체 가능
- 필요시 전용 데이터베이스 구축 가능
