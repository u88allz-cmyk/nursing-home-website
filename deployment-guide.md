## Replit 배포 방법

### 1단계: 프로젝트 확인
현재 프로젝트가 정상 작동하는지 확인하세요.
- 모든 페이지가 로드되는지 테스트
- 폼 제출이 작동하는지 확인
- 이미지가 정상 표시되는지 점검

### 2단계: 배포 시작
1. **Replit 대시보드에서 "Deploy" 탭 클릭**
2. **"Deploy Now" 버튼 클릭**
3. **배포 설정 확인:**
   - Build command: `npm run build`
   - Start command: `npm run start`
   - Port: 80 (자동 설정됨)

### 3단계: 도메인 설정
배포 완료 후:
- 자동으로 `[프로젝트명].replit.app` 도메인 제공
- 커스텀 도메인 연결 가능 (유료)

### 4단계: 환경 변수 설정 (필요시)
Contact 폼을 위한 데이터베이스 연결:
1. Replit의 "Secrets" 탭에서 환경변수 설정
2. `DATABASE_URL` 등 필요한 변수 추가
